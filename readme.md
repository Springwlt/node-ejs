# Node Ejs Api

如何使用进行模板数据渲染

## 技术栈
- nodejs
- express
- ejs

## 项目简介
- 使用ES6/ES7特性
- 使用ESlint进行语法检测，遵循[Airbnb JavaScript Style Guide](https://github.com/airbnb/javascript)

# 安装依赖
yarn install

# 运行
yarn run start

# 构建
yarn run build

# 测试
yarn run test

# 部署
yarn run deploy:dev  部署至开发服务器
yarn run deploy:prod 部署至正式服务器
yarn run stop        暂停服务
yarn run reload      重载服务
